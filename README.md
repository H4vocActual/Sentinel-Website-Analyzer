Sentinel Website Analyzer is a fast tool for analyzing the A record and IP of a website
</p>

## Features:
* Extracting A record of websites
* Extracting website provider
* ASN
* Based on RIPE database


## Usage

```
pip3 install -r requirements.txt
python3 Sentinel.py
```
